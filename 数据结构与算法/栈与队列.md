# 栈与队列

### 栈

顺序栈的实现

top指针：指示栈顶元素在顺序栈中的位置

base指针：指示栈底元素在顺序栈中的位置

stack size:表示栈的最大容量

base == top :栈空

top -base == stack size :栈满了

上溢：栈已满，还要压入元素，报错处理

下溢：栈已空，还要弹出元素，视为结束条件
